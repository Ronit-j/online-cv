Graph neural network for social navigation

Luis J. Manso Ronit Jorvekar, Pillar Bacillar,

code

SNGNN is a Graph Neural Network to model adherence to social-navigation conventions for robots. Given a particular scenario composed of a room with any number of walls, objects and people (who can be interacting with each other) the network provides a social adherence ratio from 0 to 1. This information can be used to plan paths for human-aware navigation. In recent update, we included dynamic scenarios dataset and the model was upgraded to estimate the social score based on the dynamic scenario
